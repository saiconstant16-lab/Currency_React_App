
A simple currency converter component built using React and TypeScript.

## Files
- `CurrencyConverter.tsx` – Main currency conversion logic and UI
- `CurrentCurrency.css` – Styling for the currency converter

## Features
- Convert between currencies
- Clean and simple UI
- Written using functional components
- Easy to reuse in any React project

## Usage
This component can be imported and used inside any React application.
